Cascading Style Sheet (CSS)

Introduction:

Cascading Style Sheet also known as CSS is used to design Web Pages or Websites.

↬ CSS saves time.

↬ Helps the page load faster.

↬ Easy to maintain.

↬ Independent platform.

↬ Extension: Save into .css

Types of CSS:

I. Inline CSS:

<h1 style="color: white; background-color: red;">This is Heading one.</h1>

II. Internal CSS:

<head>
    <title>CodeX</title>
    <style type="text/css">
        h1{
            color: white;
            background-color: red;
        }
    </style>
</head>

III. External CSS:

<head>
    <title>CodeX</title>
    <link rel="stylesheet" type="text/css" href="css/style.css">
</head>

4. Text Properties

h1 {
    /* 1. Text Color */
    color: red;

    /* 2. Letter Spacing */
    letter-spacing: 5px;

    /* 3. Word Spacing */
    word-spacing: 20px;

    /* 4. Text Indent (First Line Before Space Like DAWN Newspaper) */
    text-indent: 20px;

    /* 5. Text Align */
    text-align: left;
    text-align: right;
    text-align: center;
    text-align: justify;

    /* 6. Text Decoration */
    text-decoration: underline;
    text-decoration: overline;
    text-decoration: line-through;
    text-decoration: none;

    /* 7. Text Transform */
    text-transform: uppercase;
    text-transform: lowercase;
    text-transform: capitalize;

    /* 8. Text Shadow */
    /* Text Shadow: Horizental    Vertical   Blur    Color; */
    text-shadow: 2px 2px 4px gray;
}

5. Font Properties

h1 {
    /* If Font Name Contains More Than One Words Then Use Like This: "Times New Roman"*/
    font-family: Arial;

    font-style: italic;
    font-style: oblique;

    font-weight: bold;

    /* All Fonts In Capital */
    font-variant: small-caps;

    /* Space Between Lines */
    line-height: 100px;

    font-size: 100px;

    /* One Line Code: Shortcut */
    /* font: style      variant     weight      size/line       height    family */
    font: italic small-caps bolder 100px/100px Arial;
}

Variables

:Root
  • The :root class is used to define variables in CSS.
  • A variable is a data container that stores data and it is defined by :root class.
:root {
    --bgcolor: orange;
}

body {
    background-color: var(--bgcolor);
}

    background-color: var(--bgcolor);
