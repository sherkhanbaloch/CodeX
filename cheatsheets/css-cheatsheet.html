Cascading Style Sheet (CSS)

Introduction:

Cascading Style Sheet also known as CSS is used to design Web Pages or Websites.

↬ CSS saves time.

↬ Helps the page load faster.

↬ Easy to maintain.

↬ Independent platform.

↬ Extension: Save into .css

Types of CSS:

1. Inline CSS:

<h1 style="color: white; background-color: red;">This is Heading one.</h1>

2. Internal CSS:

<head>
    <title>CodeX</title>
    <style>
        h1{
            color: white;
            background-color: red;
        }
    </style>
</head>

3. External CSS:

<head>
    <title>CodeX</title>
    <link rel="stylesheet" type="text/css" href="css/style.css">
</head>

Color Property

h1 {
    color: white;
    background-color: #00ff00;
    color: rgb(255, 0, 0);
    border: 2px solid blue;
    color: rgba(255, 0, 0, 0.5); /* Alpha (Transparent) can be used from .0 to 1 */   
}
