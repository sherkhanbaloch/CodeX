Hyper Text Markup Language (HTML)

Introduction:

Hyper Text Markup Language also Know as HTML is used to create Web Pages or Websites. It is developed by Tim Berners-Lee in 1993. It is a service which runs Web Pages.

↬ Text Editiors: NotePad, Notepad++, Sublime Text, VS Code etc.

↬ Output: On any web browser.

↬ Extension: Save into .html or .htm

HTML Tags:

There are two types of tags in HTML

1. Paired Tags: Which starts and close. Like <html> </html>, <a> </a>

2. Unpaired Tags: Which starts but don't close. Like <br> <hr>, <img>

Boiler Plate Code:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CodeX</title>
</head>
<body>
    <h1>CodeX</h1>
</body>
</html>
